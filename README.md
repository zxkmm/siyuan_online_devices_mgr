# siyuan_online_devices_mgr
A powerful remote management and notification plugin for SiYuan. Get notified when your devices come online and manage them remotely from anywhere.

# Features
*   [x] **Online Notifications**: Get instant push notifications on your phone (via Bark) when your SiYuan instances start up.
*   [x] **Online Device List**: View all your online devices in real-time within a custom dock.
*   [x] **Remote Lock**: Lock the screen of your remote devices with **real-time success detection** and visual feedback.
*   [x] **Remote Access Auth Code**: Remotely set or update the Access Auth Code for other devices with a secure double-check verification.
*   [x] **Password History**: Keep a local record of passwords set from your device to prevent lockouts.
*   [x] **Remote Exit**: Gracefully shut down remote SiYuan instances.
*   [x] **Remote Clipboard**: Send text directly to the clipboard of a specific device or broadcast to all online devices.
*   [x] **Remote Messaging**: Send popup messages to remote devices.
*   [x] **Trigger Sync**: Manually trigger a cloud sync on remote devices.

# Setup Guide

## Remote Management (via GoEasy)
1.  Register a [GoEasy](https://www.goeasy.io/) account.
2.  Create a free instance.
3.  Copy the **Common Key (Client Side)** token and paste it into the plugin settings.
4.  Sync the plugin settings to all your devices.
5.  Open the Online Devices Dock to start managing!

## Online Notification (via Bark)
1.  Install [Bark](https://github.com/Finb/Bark) on your phone.
2.  In the plugin settings, enter your **Bark Server Base URL**.
3.  (Optional) Enable notifications for specific devices only using the "Device List" filter.

# Precautions
*   **Privacy First**: This plugin must **not** be used as a tool to monitor others' behavior.
*   **Data Security**: Device names and identifiers are transmitted as part of the URI. While encrypted via TLS, be aware that SaaS providers (Bark/GoEasy) may technically have access to this metadata.
*   **Legal Disclaimer**: Any data generated by this plugin is for personal convenience only and has no legal validity.
